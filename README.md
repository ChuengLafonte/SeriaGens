# SeriaGens ⚡

A high-performance, feature-rich Custom Generator Tycoon plugin for Minecraft servers (version 1.21+). Designed for **Project Seria**, this plugin offers a unique progression system with mechanical corruption, integrated economy, and seamless Skyblock (BentoBox) support.

![GitHub Version](https://img.shields.io/badge/version-1.1.0-blue)
![Java Version](https://img.shields.io/badge/java-21-orange)
![Platform](https://img.shields.io/badge/platform-Paper-green)

---

## ✨ Key Features

- 💎 **11+ Default Generator Tiers**: From common Coal to legendary Deepslate Iron.
- ⚙️ **Mechanical Corruption System**: Generators can break and require repair (active only when owner is online).
- 🚀 **Dynamic Events**: Global multipliers for Speed, Drops, Sell Value, and Generator Tiers.
- 🛍️ **Integrated GUI Shop & Management**: Manage your entire energy grid from a single menu.
- ⚡ **Sell Wands**: Customizable wands to sell chest contents with multipliers.
- 🏝️ **BentoBox Integration**: Native support for Skyblock island protection and management.
- 📊 **Energy Grid Management**: Joule-based power consumption and capacity limits.
- 📦 **NBT-Based Economy**: High-performance item selling using persistent data.

---

## 🛠️ Quick Start

1. **Install Dependencies**: Ensure you have [Vault](https://www.spigotmc.org/resources/vault.3431/) and a compatible economy plugin installed.
2. **Download**: Drop `SeriaGens.jar` into your `/plugins` folder.
3. **Configure**: Start your server to generate the default configuration files in `/plugins/SeriaGens`.
4. **Permissions**: Assign `seriagens.shop` and `seriagens.place` to your players to get started.

## 📜 Commands

| Command | Alias | Description |
| :--- | :--- | :--- |
| `/sgens help` | `/gen` | View all available player commands. |
| `/genshop` | `/gshop` | Open the generator purchasing menu. |
| `/genset` | `/mygens` | Open your generator management GUI. |
| `/gensell` | `/gsell` | Sell all generator drops in your inventory. |
| `/event` | `/events` | (Admin) Manage global generator events. |

*See [Commands & Permissions](docs/Commands-and-Permissions.md) for a full reference.*

## 🏗️ Requirements

- **Paper/Spigot**: 1.21 or higher.
- **Java**: 21+.
- **Vault**: Essential for the economy system.
- **BentoBox** (Recommended): For island protection.
- **PlaceholderAPI** (Optional): For custom placeholders.
- **FancyHolograms** (Optional): For visual corruption alerts.
